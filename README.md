# ignisBot
discord bot running on node and discord.js

Bot requires custom json files:

token.json file in /data folder:<br>
  {<br>
    "token": "discord_bot_token"<br>
  }<br>

  display.json file in /data folder:<br>
  {<br>
    "message": "DEBUG VERSION RUNNING",<br>
    "type": "PLAYING"<br>
  }<br>
